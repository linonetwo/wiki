# 寄生修仙（设定）
## 一、修炼阶段对照表
| 世人所认为的修仙阶段 | 实际上的虫族寄生阶段 |
|----------------------|----------------------|
| 炼气期               | 培肉期               |
| 筑基期               | 发育期               |
| 结丹期               | 结茧期               |
| 元婴期               | 成虫期               |
| 飞升                 | 回归（虫族母星）     |

---

## 二、各阶段详细设定
### 1. 炼气期（培肉期）
- **核心设定**：被虫族判定为适合寄生的人类，通过“修真”激发身体潜能，【真正目的】是让肉体变得更可口、更适宜孵化虫族。
- **灵根设定**：适合被虫族寄生的人类被称为“有灵根”，分为五种灵根，实则对应五大虫族的寄生适配性；不适合被虫族寄生的人类，被称为【凡人】。
- **配套机制**：五行灵根对应不同虫族的寄生适配，凡人因无法被寄生，被排除在修仙体系之外。

### 2. 筑基期（发育期）
- **筑基丹真相**：对外宣传是进入筑基期的灵丹妙药，**实际是外星虫卵**，只有被虫卵寄生的人，才能进入下一阶段的“修炼”。
- **门派筛选机制**：筑基丹极其珍贵，门派会让弟子们互相厮杀，只有最强、最优秀的弟子才能得到虫卵，这种残酷的门派厮杀，死亡率往往超过七成。
- **寄生成功率**：即便服用了筑基丹（虫卵），产生排异反应的人类也会立刻死亡，筑基成功率亦不过三成。
- **幼虫改造阶段**：
  1.  幼虫寄生后，会从人类内部开辟名为“气海”的区域，开始成长，并且改造宿主弱小的身体。
  2.  因为虫族有比人类更长的寿命，筑基成功的修仙者会变得比原本更年轻（例：60岁炼气期 → 61岁筑基期，外貌大幅年轻化）。
  3.  随着幼虫逐渐成长，宿主会觉醒强大而非人的力量，例如身体的能量变化、操作虫族科技道具（法器）。
  4.  不同虫族幼虫的特性赋予宿主不同能力：擅长喷火的幼虫赋予宿主火焰之力，擅长飞行的幼虫可令宿主长出双翼；旁人看来的“仙法不同”，实则是不同虫种导致的差异。

### 3. 结丹期（结茧期）
- **身体改造**：成功突破筑基、到达结丹期（结茧期）的修仙者（宿主），身体进一步向虫人化改造，外形异于常人。
- **认知伪装**：宿主只需用幻术稍微遮掩，加上常人默认“奇形怪状是道骨仙风”，甚至不少人认为只有长得奇形怪状，才是高阶修仙者该有的模样。
- **权力与觉醒**：结丹（结茧）期的宿主基本已被虫族控制，是门派最有用的中坚力量，替不方便出门的太上长老操纵和管理各项事务；同时，结丹期的修仙者也是最容易产生心魔（觉醒）的时期。

### 4. 元婴期（成虫期）
- **破茧成虫**：当成虫破茧而出时，宿主就进入了元婴期；修炼大成后的成虫拥有远超结丹期十数倍的力量，并且可以从宿主身上脱离。
- **宿主的结局**：成虫不会放弃宿主的身体，在上百年中被改造、拥有强横力量和神力的肉体，是虫类最好的武器和装甲。
- **完全掌控与繁衍**：成虫期（元婴期）的宿主，失去了自我意识，被虫族完全掌控，肉体也会变得更加诡异，同时会拥有虫类最重要的能力——繁衍后代。
- **虫卵生产**：成虫的力量、成长度不同，一年可以产出少则四五十、多则上百枚的虫卵（筑基丹）；虫卵是一个虫巢繁衍的关键，所以成虫会以太上长老的身份，长期闭关，进一步发育并且安心产出后代。
- **门派结构**：太上长老因为深居简出，少有人看到，看见的人亦会因为其古怪的性格、可怖的造型私下称其为“老怪物”，却不知这些尚未失去自我意志的人类，却怎知这确实就是真的怪物呢。
- **虫巢规则**：一个没有太上长老（成虫）的门派（虫巢）是没有根基的，小型门派也会有至少一位元婴期的太上长老，中型虫巢有3~5只，大型虫巢则有10只以上。
- **门派战争**：虽然同为虫类，但不同门派因为族群不同，又互相争夺修炼资源、洞天福地，会互相争斗杀伐；只是这样的战斗只会发生在低阶，维持的生成的战斗会互相受益进化，从而产生了修仙界“太上长老不得出手”的规则。

### 5. 飞升（回归虫群）
- **飞升真相**：漫长的成长期完毕后，元婴修士（成虫）会一飞而起，破碎虚空，去到传说中的灵界；**实际上**，是离开了孕育自己的星球，去往虫族的母星。
- **母星归宿**：在母星，它会回归自己的族群，回到自己的母虫身边，但这并不是结束；它会以下界修士（飞升虫族）的名义，为了虫群，参与到更多的战斗中去。
- **飞升虫族的强度**：这些生长周期漫长的寄生虫族，体内蕴含着人类与虫类共生的巨大力量，虽然它们中极少才能成长到飞升期，但每个飞升虫族，都拥有远超在宇宙中长大的同阶无数倍的韧性、狡猾和战斗经验，是人类帝国最不想面对的棘手敌人。

## 三、世界观设定
### 1. 三千世界
- 这样的修仙星球，已经完全是虫族用来制造飞升虫类的巢穴；而这样的星球，被虫族占据了不下数千个，又被称为**三千世界**。
- 这些星球不仅是虫族的兵营，无数年来，帝国的勇士们一直想夺回这些星球，拯救星球上的人民。

### 2. 人类远征军的困境
- 只有最勇敢、最愚蠢的战士，才敢突破重围，杀到三千世界去。
- 在这些被虫族控制的星球上，勇士的义举却不会被愚民所理解：整颗星球上，所有的修仙者、所有的名门正派，完全被虫控制的，都是敌人。
- 民众认知：人类远征军会被修仙者和凡人称为“堕仙”，太上长老会下令“诛堕仙者，赏长生丹，高阶法宝，开门立派！”，全民喊出“杀堕仙！”的口号，将拯救者视为入侵的敌人。

## 四、剧情钩子（同人创作开篇）
一名人类远征军的机甲战士，迫降在修仙星球，误入一处洞府，遇到了一群觉醒了自我意识、反抗虫族的“邪修”。
- 邪修首领：“没想到我这小小洞府，居然会有上界的堕仙大驾光临”
- 战士：“你们是谁！”
- 邪修首领：“别紧张，道友，我们只是暂居此处的一群…区区邪修而已，追杀道友的虫类爪牙，我们已经诛杀了。只是不知道友是否有兴趣……一起把这个丑陋不堪的修仙界，搅个天翻地覆呢？”

## 五、核心设定补充
- **核心反转**：修仙体系本质是虫族的寄生养殖流程，所有“修炼”都是为了培育适合寄生、孵化、成长的宿主，最终产出强大的飞升虫族。
- **阶层逻辑**：从炼气到飞升，是宿主被寄生、改造、控制、异化、最终完全成为虫族工具的完整流程，段位越高，寄生越彻底，人类意识越被抹杀。
- **冲突核心**：人类帝国vs虫族、觉醒的反抗者vs被控制的修仙者、拯救者vs被愚弄的凡人，三重矛盾构建核心剧情。
